skelhs by basbossink

A PasteScript template for a simple Haskell module including HUnit and QuickCheck tests.

Skelhs is simple code generator utilising PasteScript to generate the boilerplate contents of a Haskell module and the associated test module.

Dependencies

Python PasteScript Cheetah

Install

# @git clone git://github.com/basbossink/skelhs.git@ # @cd skelhs.basicmodule@ # @python setup.py install@ ## Verify your install with: @paster create --list-templates@ The list of available templates should contain the following line: @skelhs_basic_module: A haskell module with a test environment@
